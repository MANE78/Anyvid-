# مداد الهدي - موقع القرآن الكريم

## 🌟 نظرة عامة

مداد الهدي هو موقع شامل للقرآن الكريم مبني بتقنيات حديثة، يوفر تجربة متميزة لقراءة والاستماع وتدبر القرآن الكريم مع التفسير والترجمة والتلاوات المتنوعة.

## ✨ الميزات

### 📖 القراءة والتصفح
- عرض جميع سور القرآن الكريم (114 سورة)
- نص عربي واضح بخط مميز
- تصفح سهل ومنظم للآيات
- أرقام الآيات باللغة العربية

### 🎧 التلاوة والاستماع
- مجموعة واسعة من القراء المشهورين
- تلاوات عالية الجودة
- تحكم في سرعة التشغيل
- تشغيل تلقائي اختياري

### 🔍 البحث المتقدم
- بحث في النص العربي
- بحث في الترجمات المختلفة
- نتائج مفصلة مع السياق
- فلترة متقدمة للنتائج

### 📚 التفسير والترجمة
- تفاسير متعددة
- ترجمات بلغات مختلفة
- عرض جنباً إلى جنب
- إخفاء وإظهار حسب الحاجة

### ❤️ المفضلة والإعدادات
- حفظ الآيات المفضلة
- حفظ السور المفضلة
- حفظ التلاوات المفضلة
- إعدادات شخصية مفصلة

### 🎨 التصميم والتجربة
- تصميم متجاوب لجميع الأجهزة
- وضع داكن وفاتح
- خطوط عربية مخصصة
- واجهة سهلة الاستخدام

## 🚀 التقنيات المستخدمة

- **Frontend Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom Components
- **Icons**: Lucide React
- **Animation**: Framer Motion
- **HTTP Client**: Axios

## 📡 APIs المستخدمة

### 1. AlQuran Cloud API
- **URL**: `https://api.alquran.cloud/v1`
- **الاستخدام**: النصوص العربية، الترجمات، البحث
- **الحالة**: ✅ يعمل

### 2. MP3 Quran API
- **URL**: `https://mp3quran.net/api/v3`
- **الاستخدام**: قائمة القراء، التلاوات الصوتية
- **الحالة**: ✅ يعمل

### 3. Quran.com API
- **URL**: `https://api.quran.com/api/v4`
- **الاستخدام**: التفاسير، معلومات إضافية
- **الحالة**: ✅ يعمل

## 🛠️ التثبيت والتشغيل

### متطلبات النظام
- Node.js 18.0 أو أحدث
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone https://github.com/username/madad-alhuda.git
cd madad-alhuda
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **تشغيل الخادم المحلي**
```bash
npm run dev
```

4. **بناء للإنتاج**
```bash
npm run build
```

5. **تشغيل النسخة المبنية**
```bash
npm start
```
